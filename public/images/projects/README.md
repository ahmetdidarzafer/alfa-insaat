# Proje Görselleri — GEÇİCİ STOK FOTOĞRAFLAR

Bu klasördeki `project-1/2/3.jpg` dosyaları ve `public/images/hero.jpg`
şu an **telifsiz stok mimari fotoğraflardır** (picsum.photos / Unsplash
kaynaklı, geçici). Stüdyonun gerçek proje fotoğrafları temin edildiğinde
aynı adlarla üzerine yazılmalıdır:

| Dosya | Proje | Önerilen boyut |
|---|---|---|
| `project-1.jpg` | Villa Renovasyonu | 1200×900 (4:3) |
| `project-2.jpg` | Ofis Yenileme | 1200×900 (4:3) |
| `project-3.jpg` | Rezidans İç Mimari | 1200×900 (4:3) |
| `../hero.jpg` | Ana sayfa hero görseli | 1200×1500 (4:5, dikey) |

Notlar:

- Görseller `next/image` ile `object-cover` olarak kullanılıyor; farklı
  oranlı görseller kırpılarak gösterilir.
- Gerçek fotoğraflar geldiğinde dosya adları değişmezse kodda değişiklik
  gerekmez.
